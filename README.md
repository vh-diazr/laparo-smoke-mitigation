# Non-homogeneous Surgical Smoke Mitigation in Laparoscopic Images Using a Hybrid Physical-Neural Model

> **Manuscript under review** — *International Journal of Computer Assisted Radiology and Surgery* (IJCARS), Springer Nature.
>
> Code, dataset, and model weights will be made publicly available upon acceptance.
> This repository is currently **private** and accessible only to authorized personnel during the peer-review process.

## Abstract

Minimally Invasive Surgery relies on laparoscopic imaging to provide indirect visualization of the surgical scene. A major challenge in this procedure is surgical smoke generated by electrocautery instruments, which degrades visibility and compromises clinical interpretability. This work presents a hybrid learning-based and atmospheric optics–driven method for non-homogeneous surgical smoke mitigation in laparoscopic images.

The proposed method adopts a two-stage strategy: (1) a hybrid Convolutional Neural Network and Vision Transformer (CNN-ViT) architecture jointly estimates the ambient light vector and the spatially varying transmission function of the scene from a degraded laparoscopic frame; (2) a physics-based image restoration model recovers the smoke-free image using the obtained estimates. The method is trained on a synthetic dataset constructed via a physics-based Navier–Stokes fluid simulation of surgical smoke dynamics.

Quantitative evaluation on ten independent laparoscopic sequences demonstrates consistent outperformance over four state-of-the-art baseline methods — DuRN-US, DehazeFormer, MixDehazeNet, and Desmoke-GAN — across all sequences in both PSNR and SSIM. Generalization to real laparoscopic sequences degraded by electrocautery smoke is also demonstrated.

## Method Overview

### Physical Image Formation Model

Surgical smoke degradation is modelled using the atmospheric scattering equation:

```
f(x,y) = J(x,y) · T(x,y) + A · (1 - T(x,y))
```

where `f(x,y)` is the observed degraded image, `J(x,y)` is the latent smoke-free scene, `T(x,y) ∈ [0,1]` is the spatially varying transmission map (T → 0: dense smoke; T → 1: clear), and `A ∈ ℝ³` is the global ambient light vector. Image restoration is achieved by inverting this model given estimates of `T̂` and `Â`.

### Hybrid CNN-ViT Architecture

```
Input degraded image f(x,y)  [N × M × 3]
        │
        ▼
┌─────────────────────┐
│   CNN Encoder        │  4 stages: {32, 64, 128, 256} channels
│   (downsampling ×16) │  Conv(3×3) + BN + ReLU + StrConv(2×2)
└────────┬────────────┘
         │  F ∈ ℝ^{256 × N/16 × M/16}
         ├─────────────────────────────────────┐
         ▼                                     ▼
┌────────────────────┐              ┌──────────────────────┐
│  Vision Transformer │              │  Ambient Light Head   │
│  (L=4 layers, MHSA) │              │  (MLP regression)     │
│  Self-attention      │              │  Â ∈ ℝ³              │
└────────┬───────────┘              └──────────────────────┘
         │  F_T ∈ ℝ^{256 × N/16 × M/16}
         ▼
┌─────────────────────┐
│   CNN Decoder        │  Skip connections from encoder
│   (upsampling ×16)   │  Transposed convolutions
└────────┬────────────┘
         ▼
  T̂(x,y) ∈ [0,1]    (transmission map estimate)
         │
         ▼
┌─────────────────────────────────────────┐
│   Physics-based Image Restoration        │
│   Ĵ(x,y) = (f(x,y) - Â) / T̂(x,y) + Â  │
└─────────────────────────────────────────┘
         │
         ▼
  Ĵ(x,y)   (restored smoke-free image)
```

The encoder extracts hierarchical local features through strided convolutions, progressively downsampling the spatial resolution by a factor of 16. The Vision Transformer (ViT) module captures long-range spatial dependencies among encoder tokens through multi-head self-attention (MHSA), producing globally contextualized feature representations. The decoder reconstructs the full-resolution transmission map via transposed convolutions with skip connections from the encoder. An auxiliary prediction head regresses the global ambient light vector `Â` from the encoder feature map.

## Dataset

### Synthetic Training Dataset

The training corpus was constructed by applying a physics-based surgical smoke simulation to undegraded laparoscopic sequences sourced from the [EndoScapes dataset](https://endoscapes.grand-challenge.org/). The simulation is grounded in a 2D numerical solver of the Navier–Stokes fluid dynamics equations, producing non-homogeneous and temporally dynamic smoke patterns representative of real intraoperative conditions.

| Property | Value |
|---|---|
| Source sequences (training) | 30 laparoscopic videos (EndoScapes) |
| Total training triplets | 9,460 (degraded frame, ground-truth frame, transmission map) |
| Image resolution | 512 × 480 pixels |
| Normalization | [0, 1], 256 quantization levels |
| Smoke conditions | Localized dense clouds to spatially diffuse distributions |

Each training sample is a triplet `{f(x,y), J(x,y), T(x,y)}` consisting of: a synthetically degraded frame, its corresponding ground-truth smoke-free frame, and the ground-truth transmission map generated by the simulation.

### Evaluation Dataset

Quantitative performance evaluation was conducted on ten independent laparoscopic sequences not included in the training or validation sets, extracted from the [Cholec80 dataset](https://github.com/CAMMA-public/cholec80).

| Property | Value |
|---|---|
| Source sequences (evaluation) | 10 laparoscopic sequences (Cholec80) |
| Total evaluation frames | 1,981 synthetically degraded frames |
| Procedure type | Cholecystectomy |
| Sequence duration | ~40 seconds each |

## Surgical Smoke Simulation

The physics-based simulation (`simulation/fluid2d_smoke_fast_corregida_.py`) implements a 2D numerical solver for the incompressible Navier–Stokes equations to model the fluid dynamics of surgical smoke within the insufflated abdominal cavity. The simulation parametrizes:

- Spatial onset and duration of smoke emission (associated with surgical instrument location)
- 2D emission origin and initial plume radius
- Smoke density, buoyancy, and vorticity parameters
- Temporal evolution of the transmission function `T(x,y,t)`

This approach enables the generation of physically consistent, spatially non-homogeneous smoke distributions that closely replicate the optical degradation observed during real electrocautery procedures.

## Training Protocol

### Loss Function

The model is optimized by minimizing a composite loss function that combines:

1. **Transmission loss** (`L_T`): pixelwise ℓ₁ loss between the estimated and ground-truth transmission maps
2. **Restoration loss** (`L_R`): Structural Similarity Index Measure (SSIM) loss between the restored and ground-truth images

```
L = λ_T · L_T + (1 - λ_T) · L_R
```

where `λ_T` is annealed from 0.7 to 0.3 using a cosine schedule over training epochs, progressively shifting optimization emphasis from transmission estimation to perceptual image restoration quality.

### Optimization

| Hyperparameter | Value |
|---|---|
| Optimizer | Adam |
| Learning rate | 10⁻³ (constant, 5-fold CV) |
| Scheduler | Warmup + Cosine Annealing (full training) |
| Batch size | 12 |
| Epochs (5-fold CV) | 80 |
| Epochs (full training) | 120 (early stopping, patience=25 on val. SSIM) |
| Embed dimension | 256 |
| Transformer layers | 4 |
| Training/Validation split | 90% / 10% (video-level, no temporal leakage) |

### Data Augmentation

Online augmentation was applied during training: random 180° rotations, horizontal flips, and vertical flips. Augmentations preserving the original 480×512 spatial dimensions were exclusively employed to maintain compatibility with the fixed-resolution model architecture.

## Results

### Five-Fold Cross-Validation

Internal generalization was assessed via five-fold cross-validation (5-fold CV) with video-level partitioning to prevent data leakage across folds.

| Metric | Training | Validation |
|---|---|---|
| Loss | 0.0340 ± 0.0008 | 0.040 ± 0.004 |
| SSIM | 0.9556 ± 0.0009 | 0.951 ± 0.005 |
| PSNR (dB) | 34.25 ± 0.16 | 32.17 ± 0.59 |

Low inter-fold variance confirms stable convergence and limited sensitivity to data partitioning, with no evidence of overfitting.

### Quantitative Comparison on Unseen Sequences

The proposed method was compared against four baseline methods on ten independent evaluation sequences:

| Method | PSNR (dB) | SSIM |
|---|---|---|
| **Proposed (CNN-ViT)** | **30.77 ± 4.81** | **0.924 ± 0.045** |
| DehazeFormer-S | 21.56 ± 3.57 | 0.856 ± 0.057 |
| DuRN-US | 20.59 ± 4.95 | — |
| MixDehazeNet-L | 19.06 ± 2.74 | — |
| Desmoke-GAN | 18.91 ± 3.14 | — |

PSNR improvements over the best-performing baseline range from **+7.32 dB** (Sequence 2) to **+11.24 dB** (Sequence 3). Statistical significance was confirmed via the Wilcoxon signed-rank test (W=0, p=0.002), with Cohen's d effect sizes exceeding 0.8 in all pairwise comparisons (large effect threshold).

The proposed method additionally demonstrates lower intra-sequence temporal SSIM variability (mean σ = 0.037) compared to all baselines (range: 0.048–0.054), indicating superior temporal consistency in restored sequences — a property of direct clinical relevance for intraoperative use.

## Usage

### Inference on a Single Image

```python
import torch
import cv2
import numpy as np
from model.hybrid_cnn_vit_updated import HybridCNNTransformer

# Load model
device = torch.device("cuda" if torch.cuda.is_available() else "cpu")
model = HybridCNNTransformer(img_hw=(480, 512), embed_dim=256, num_layers=4).to(device)
model.load_state_dict(torch.load("weights/model_weights.pth", map_location=device))
model.eval()

# Load and preprocess degraded image
img = cv2.imread("path/to/degraded_image.png").astype(np.float32) / 255.0
img_tensor = torch.from_numpy(img.transpose(2, 0, 1)).unsqueeze(0).to(device)

# Inference
with torch.no_grad():
    T_pred, A_pred = model(img_tensor)
    A_pred = A_pred[:, :, None, None]
    J_restored = (img_tensor - A_pred) / T_pred.clamp(min=1e-3) + A_pred
    J_restored = J_restored.clamp(0.0, 1.0)

# Save restored image
output = (J_restored.squeeze().cpu().numpy().transpose(1, 2, 0) * 255).astype(np.uint8)
cv2.imwrite("restored_image.png", output)
```
